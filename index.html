Food Inflation Report

This project analyzes average inflation and annual food inflation in various countries, as we l as their impact during the COVID-19 pandemic. Data from this report is processed to produce visualizations that provide insight into economic stability, pressure on the food sector due to inflation, and comparisons of economic performance between countries. The results of this visualization can be used as a tool for economic policy analysis and strategic planning at national and international levels.

Stunting Data Analysis in Indonesia

This project contains analysis and visualization of data about stunting cases in Indonesia from 2021 - 2023. In this project, data was taken from one of the websites using Web Scrapping. According to WHO (2015), stunting is a growth and development disorder in children due to chronic malnutrition and recurring infections, characterized by height below the standard.

An IoT-based plant monitoring system that uses a DHT11 sensor to measure air temperature and humidity, as well as a soil moisture sensor to monitor soil conditions. Sensor data is displayed on the LCD and sent to the ThingSpeak platform for remote monitoring via a web interface.

Bicycle Rental Data Analysis & Visualization

This project contains analysis and visualization of data about bicycle rental. This dashboard was created using Streamlit and contains the total number of bicycle renters, average daily bicycle rentals, average bicycle rentals by season, bicycle rental trends by hour, and comparison of casual vs registered user bicycle rentals.

Easy Tour

Developed a content-based filtering recommendation system using TensorFlow to analyze and predict the similarity between tourist places based on their categories. Also created a Flask REST API to handle user requests and provide recommendations. The system includes a TF-IDF vectorizer to convert text features into numerical vectors, a neural network model to classify places into categories, and cosine similarity calculations to determine the most similar places.

IMDb Sentiment Analysis Review

This project contains an end-to-end machine learning pipeline using TensorFlow Extended (TFX) for IMDB movie review sentiment classification. The pipeline automatically processes IMDB review data, performs text preprocessing, builds and trains a neural network model, evaluates model performance, and deploys the model for inference. The system is able to classify reviews as positive or negative sentiment with a fairly high level of accuracy.
